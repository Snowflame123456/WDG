## 1. CONCEPTOS BÁSICOS

### ¿Qué tipos de agua hay que gestionar en una casa?

En Australia, hay **tres sistemas de agua** completamente separados:

```
┌─────────────────────────────────────────────────────────────────┐
│  SISTEMA 1: STORMWATER (Aguas Pluviales)                        │
│  → Lluvia que cae en el techo, jardín, entrada de autos         │
│  → Va hacia la calle o sistema de stormwater del council        │
│                                                                 │
│  SISTEMA 2: SEWER (Aguas Servidas / Cloacas)                    │
│  → Agua de baños, cocina, laundry                               │
│  → Va hacia la red cloacal de South East Water                  │
│                                                                 │
│  SISTEMA 3: OSD (On-Site Detention)                             │
│  → Tanque que retarda la descarga de stormwater                 │
│  → Requerido en muchas subdivisiones nuevas                     │
└─────────────────────────────────────────────────────────────────┘
```

> ⚠️ **REGLA DE ORO:** Stormwater y Sewer NUNCA se mezclan. Son sistemas separados e incompatibles. Mezclarlos es una violación grave de las normas australianas.

## 4. ANÁLISIS DE PENDIENTES

### PASO 2 — Entender las Pendientes del Terreno

#### 4.1 ¿Qué es la Pendiente y por qué importa?

La pendiente (slope o gradient) determina hacia dónde fluye el agua naturalmente. Se expresa en porcentaje (%) o como ratio (1:X).

```
FÓRMULA:
Pendiente (%) = (Diferencia de altura / Distancia horizontal) × 100

EJEMPLO:
Si el terreno cae 0.5m en 10m de distancia:
Pendiente = (0.5 / 10) × 100 = 5%
```

#### Tabla 3: Pendientes Mínimas Según AS/NZS 3500.3

| Elemento | Pendiente Mínima | Pendiente Recomendada | Notas |
|---|---|---|---|
| Canaletas de techo (gutters) | 1:500 (0.2%) | 1:300 (0.33%) | AS/NZS 3500.3 Cl. 3.5 |
| Tuberías de PVC enterradas DN100 | 1:100 (1%) | 1:80 (1.25%) | AS/NZS 3500.3 Tabla 3.1 |
| Tuberías de PVC enterradas DN150 | 1:150 (0.67%) | 1:100 (1%) | AS/NZS 3500.3 Tabla 3.1 |
| Surcos/canales superficiales | 1:200 (0.5%) | 1:100 (1%) | Velocidad mínima 0.6 m/s |
| Drenes franceses (agricultural) | 1:200 (0.5%) | 1:100 (1%) | Para auto-limpieza |
| Sewer — tuberías DN100 | 1:60 (1.67%) | 1:40 (2.5%) | AS/NZS 3500.2 |

> ⚠️ **IMPORTANTE:** Una pendiente mayor al 10% (1:10) en tuberías puede causar problemas de separación de líquidos y sólidos. Consultar engineer si el terreno es muy empinado.

#### 4.2 Cómo Leer el Survey y Determinar Pendientes

**PASO 2A — Identificar las Curvas de Nivel (Contour Lines):**

Las curvas de nivel en el plano topográfico conectan puntos de igual altura. Cuando están muy juntas = terreno empinado. Cuando están separadas = terreno plano.

```
EJEMPLO DE LECTURA DE COTAS EN UN LOTE DE HEIDELBERG HEIGHTS:

     CALLE (Street Level)
     RL = 75.200m AHD
         │
         │ ← Vía pública
    ─────┼──────────────────────────────
         │
    [FRENTE DEL LOTE]
    RL = 74.800m AHD
         │
         │         ←── Pendiente: 0.4m / 5m = 8% (hacia la calle)
         │
    [CENTRO DEL LOTE]
    RL = 74.500m AHD
         │
         │         ←── Pendiente: 0.3m / 15m = 2% (hacia el frente)
         │
    [FONDO DEL LOTE]
    RL = 74.100m AHD
         │
    [LINDE POSTERIOR]
```

**PASO 2B — Identificar el Punto Natural de Descarga (Natural Drainage Path):**

El agua siempre fluye hacia los puntos más bajos. Identificar:
- ¿El lote drena hacia la calle? (lo más común y deseable)
- ¿El lote drena hacia el fondo? (requiere sistema especial)
- ¿El lote tiene zonas bajas internas (sumideros)? (problem areas)

**PASO 2C — Calcular el Área de Captación (Catchment Area):**

```
Para un lote de 590m²:
├── Techo de la casa: aproximar 55% del lote = ~325m²
├── Áreas pavimentadas: driveways, paths = ~80m²
├── Jardín permeable: ~185m²
└── TOTAL IMPERMEABLE = ~405m² (importante para calcular caudal)
```

## 5. STORMWATER DRAINAGE

### PASO 3 — Diseñar el Sistema de Aguas Pluviales

#### 5.1 Calcular el Caudal de Diseño (Design Flow)

Usamos el **Método Racional** (Rational Method), el más simple y aceptado para lotes residenciales pequeños en Australia.

```
FÓRMULA DEL MÉTODO RACIONAL:
Q = C × I × A / 360

Donde:
Q = Caudal de diseño (litros por segundo, L/s)
C = Coeficiente de escorrentía (runoff coefficient) — sin unidades
I = Intensidad de lluvia (mm/hora) para el período de retorno
A = Área de captación (hectáreas)
```

#### Tabla 4: Coeficientes de Escorrentía (C) para Heidelberg Heights

| Tipo de Superficie | Coeficiente C | Notas |
|---|---|---|
| Techo (cualquier material) | 0.90 | AS/NZS 3500.3 Tabla B1 |
| Concreto / Asfalto (driveway) | 0.85 | |
| Piedras/adoquines permeables | 0.40–0.60 | Según densidad |
| Césped sobre suelo arcilloso | 0.35 | Típico en Heidelberg Heights |
| Césped sobre suelo arenoso | 0.20 | |
| Jardín con plantas densas | 0.20–0.30 | |
| Lote mixto residencial | 0.55–0.65 | Valor típico |

**Períodos de retorno que usa AS/NZS 3500.3:**

- **Sistema mayor (overland flow):** ARI 100 años (major system — no debe entrar a viviendas)

#### 5.2 EJEMPLO PRÁCTICO — Cálculo de Caudal para Nuestro Lote de 590m²

```
DATOS:
Área techo = 200 m² (casa típica 3 hab.) = 0.02 ha
Área driveway = 40 m²                    = 0.004 ha
Área jardín = 350 m²                     = 0.035 ha

CAUDALES (sistema menor = ARI 5 años, I = 80 mm/h):

Q_techo    = 0.90 × 80 × 0.02 / 360 = 0.004 m³/s = 4.0 L/s
Q_driveway = 0.85 × 80 × 0.004 / 360 = 0.00076 m³/s = 0.75 L/s
Q_jardín   = 0.35 × 80 × 0.035 / 360 = 0.00272 m³/s = 2.72 L/s

CAUDAL TOTAL = 4.0 + 0.75 + 2.72 = 7.47 L/s ≈ 7.5 L/s

→ Una tubería DN100 con pendiente 1:80 tiene capacidad ~8-10 L/s ✓
→ Una tubería DN150 con pendiente 1:100 tiene capacidad ~20 L/s ✓
```

### PASO 4 — Diseñar Bajantes (Downpipes)

#### 5.3 Bajantes — ¿Qué son y cómo funcionan?

Los bajantes (downpipes o rainwater leaders) llevan el agua desde las canaletas del techo hacia el suelo.

```
TECHO
  │
  ▼
[CANALETA] ──────────────────────────────── [CANALETA]
  │
  ▼
[BAJANTE] ← esto es el downpipe (tubo vertical)
  │
  ▼
[PIT / CONNECTION] ← conexión al sistema subterráneo
```

#### PASO 4A — Ubicación de Bajantes en el Plano

**Reglas para ubicar bajantes:**

1. **Máximo 12m de longitud de canaleta** antes de un bajante (para evitar desbordamiento)
2. **Nunca en esquinas internas** del techo (agua se acumula)
3. **Preferir esquinas externas** o paredes divisorias
4. **Distancia mínima a vecinos:** no descargar agua hacia propiedad lindante
5. **Acceso para mantenimiento:** visible y accesible

```
PLANO ESQUEMÁTICO — UBICACIÓN DE BAJANTES
(Vista desde arriba — Top View)

         CALLE

    │      TECHO                │
    │      PRINCIPAL            │
```

### PASO 5 — Sistema de Pits y Rejillas

#### 5.4 ¿Qué son los Pits y Rejillas?

Los **pits** (pozos de inspección o sumideros) son cajas enterradas que conectan diferentes partes del sistema de stormwater. Las **rejillas (grates)** cubren los pits que colectan agua superficial.

1. **En cada cambio de dirección** (curva > 45°) colocar un IO o pit

### PASO 6 — Canales y Cunetas Superficiales (Swales y Channels)

#### 5.5 ¿Qué son y cuándo se usan?

Los **swales** son canales poco profundos con forma de "V" o trapezoidal cubiertos de césped. Son alternativas económicas a las tuberías en ciertas zonas.

```
SWALE (Canal de Pasto) — Vista en Corte:

        Nivel del suelo
    ____/ \____
   /           \   ← Bordes inclinados (1:3 o más suave)
  /    Swale    \
 /_______________\
         ↓
    Fondo del canal
```

#### Tabla 11: Swale vs Tubería Enterrada — ¿Cuándo usar cada uno?

| Criterio | Swale (Canal de Pasto) | Tubería Enterrada |
|---|---|---|
| Costo | Bajo (solo excavación + pasto) | Mayor (materiales + mano de obra) |
| Mantenimiento | Cortar el pasto regularmente | Mínimo, limpiar cada 5-10 años |
| Velocidad permitida | 0.6 – 2.5 m/s | No aplica |
| Pendiente mínima | 0.5% (1:200) | 1% (1:100) para DN100 |
| Pendiente máxima | 5% sin revestimiento | Sin límite práctico |
| Funciona como filtro | SÍ (Water Sensitive Urban Design) | NO |
| Permite infiltración | SÍ | NO |
| Requiere espacio | SÍ — mínimo 0.6m de ancho | Mínimo 0.3m de profundidad |
| ¿Cuándo usar? | Jardines, lotes con espacio | Bajo pavimento, cuando no hay espacio |
| ¿Cuándo NO usar? | Bajo driveways, < 0.5% pendiente | Cuando suelo es muy permeable (usar drenes franceses) |

## 6. OSD — ON-SITE DETENTION

### PASO 7 — ¿Qué es el OSD y cuándo necesito uno?

#### 6.1 Concepto de OSD

El **On-Site Detention (OSD)** es un tanque o celda subterránea que almacena temporalmente el agua de lluvia y la libera lentamente a la red de stormwater del council, para no aumentar el caudal en las cañerías existentes.

```
SIN OSD:
LLUVIA → TECHO → CANALETA → BAJANTE → RED → (posible inundación aguas abajo)
              ↑ todo el caudal va inmediatamente

CON OSD:
LLUVIA → TECHO → CANALETA → BAJANTE → [TANQUE OSD] → Libera lentamente → RED
                                          ↑
                              Retiene agua temporariamente
                              Se libera en horas, no minutos
```

#### 6.2 ¿Cuándo es Obligatorio el OSD en Heidelberg Heights?

**En el City of Banyule, el OSD es generalmente requerido cuando:**

| Condición | ¿Requiere OSD? |
|---|---|
| Casa nueva en lote existente (no subdivisión) | Depende del council — consultar |
| Subdivisión nueva (lote nuevo) | SÍ — generalmente obligatorio |
| Ampliación que aumenta área impermeable > 50m² | Depende — consultar council |
| Demolición y reconstrucción | SÍ si el nuevo techo es mayor |
| Lote en zona con cañerías de stormwater existentes con restricciones | SÍ |

> ⚠️ **SIEMPRE consultar al City of Banyule Engineering Department antes de asumir si necesitas OSD.**  
> Hay un formulario de "Pre-Application Meeting" disponible en su página web.

#### 6.3 Cómo se Diseña un OSD (Método Simplificado)

El diseño exacto requiere un Hydraulic Engineer, pero estos son los conceptos:

**Parámetros clave del OSD:**

```
1. PERMISSIBLE SITE DISCHARGE (PSD):
   Caudal máximo que el lote puede descargar al council
   Unidad: litros por segundo (L/s)
   Para Banyule: típicamente entre 5 y 20 L/s según ubicación
   → Consultar al council para obtener el PSD de tu lote específico

2. SITE STORAGE REQUIREMENT (SSR):
   Volumen de agua que el tanque debe almacenar
   Unidad: metros cúbicos (m³) o litros
   Se calcula mediante software especializado (DRAINS, XP-SWMM, etc.)

3. ORIFICE SIZE:
   Diámetro del orificio de salida del OSD
   Controla cuánta agua sale por segundo
   Calculado para no exceder el PSD
```

#### Tabla 12: Tamaños Típicos de OSD para Lotes Residenciales en Melbourne

| Área Impermeable del Lote | SSR Típico | Dimensiones Típicas (Tanque Plástico) |
|---|---|---|
| < 200 m² | 1.5 – 3.0 m³ | 1 tanque de 2,000L |
| 200 – 350 m² | 3.0 – 6.0 m³ | 2 tanques de 3,000L |
| 350 – 500 m² | 6.0 – 10.0 m³ | Celda de hormigón o 3-4 tanques |
| > 500 m² | > 10.0 m³ | Diseño específico por engineer |

> 📌 Estos son valores indicativos. El diseño real lo hace un Hydraulic Engineer usando software DRAINS (http://watercom.com.au/drains/)

#### 7.5 Trampas (Traps) — Obligatorias en AS/NZS 3500.2

Las trampas crean un sello de agua que impide que los gases del sewer entren a la vivienda.

```
P-TRAP (más común):

     AGUA ──────┐
                │ ← Agua fluye hacia abajo
                │
          ┌─────┘
          │    ← Sello de agua (50mm mínimo)
          │
          └──── HACIA EL DRAIN
```

#### Tabla 15: Profundidad del Sello en Trampas

| Tipo de Fixture | Profundidad Mínima del Sello | Norma |
|---|---|---|
| Lavamanos, bañera, ducha | 50 mm | AS/NZS 3500.2 Cl. 9.2 |
| Inodoro (WC) — tiene trampa integrada | Interna — no modificar | — |
| Floor waste | 50 mm | AS/NZS 3500.2 Cl. 9.2 |
| Kitchen sink (bajo mesada) | 50 mm | AS/NZS 3500.2 Cl. 9.2 |

#### 7.6 Ventilación del Sistema de Sewer

**¿Por qué se necesita ventilación?**
Sin ventilación, el flujo de agua en los tubos crea vacíos de presión que "sorben" los sellos de las trampas, dejando entrar gases tóxicos y malolientes del sewer.

```
SISTEMA DE VENTILACIÓN TÍPICO:

TECHO
  │
  │ ← Stack Vent (tubo vertical que sale al exterior)
  │   DN 50 mm mínimo
  │
[STACK / RISER]
  │
  │ ← Todos los fixtures conectan aquí
  │
[HOUSE DRAIN]
```

**Reglas de ventilación (AS/NZS 3500.2):**
- El stack principal debe ventilarse a la atmósfera
- El tubo de ventilación debe terminar a **mínimo 900mm sobre el nivel del techo**
- Distancia mínima de **3m de ventanas o aberturas**
- Diámetro mínimo: **50mm** (o igual al stack, lo que sea mayor)

#### 7.7 Inspection Openings (IOs) en Sewer

**Obligatorio en:**
- Cada cambio de dirección > 45°
- Cada 12m máximo en tramos rectos
- Dentro de 12m del punto de conexión a la red pública
- En la base de cada stack vertical

## 💡 CONSEJOS FINALES PARA NO INGENIEROS

1. **Contratá siempre a un Licensed Plumber** — no es opcional en Victoria. El trabajo sin licencia es ilegal y puede invalidar tu seguro.

2. **Para OSD, contratá un Hydraulic Engineer** — son los que entienden de caudales y diseño de tanques. Buscalos en Engineers Australia.

3. **Llamá al council ANTES de diseñar** — el City of Banyule tiene ingenieros disponibles para consultas previas (pre-application meeting). Así sabrás exactamente qué necesitás.

4. **Dial Before You Dig siempre** — es gratis, es rápido (www.1100.com.au) y puede salvarte de romper una tubería de gas o electricidad.

5. **Separar stormwater de sewer** — esta es la regla más importante. Las sanciones por contaminar el sistema de sewer con stormwater son muy severas.

6. **Mantén el agua en tu propiedad** — no podés descargar stormwater hacia la propiedad del vecino. Toda el agua debe ir hacia tu Legal Point of Discharge.

7. **Documentá todo** — guardá todos los permisos, planos y certificados. Se necesitan cuando vendés la propiedad.
